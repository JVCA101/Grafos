Os grafos estão separados em dois tipos de instâncias: grafos direcionados e não direcionados. As instâncias são nomeadas seguindo o padrão (#nós)n(D para direcionados e U para não direcionados).dat

Na primeira linha de cada arquivo temos o tamanho da instância (número de nós). Da segunda linha em diante, temos o SOURCE NODE - TARGET NODE - EDGE WEIGHT nessa ordem.
